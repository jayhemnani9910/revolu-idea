CAG Deep Research System

AI-powered research automation using LangGraph orchestration, multi-source web search, and iterative verification loops to produce structured, cited reports.

Python 3.10+ LangGraph LangChain Tavily + Exa

If you just enabled GitHub Pages and still see a 404, wait ~1–2 minutes and refresh.

What it does

  • Plans a research strategy for your question.
  • Searches the web across multiple providers (Tavily + Exa).
  • Extracts and synthesizes key information.
  • Builds a lightweight knowledge graph of entities and relationships.
  • Generates a structured report and runs audit-style verification passes.

Quick start

Clone, install, configure keys, run:

git clone https://github.com/jayhemnani9910/revolu-idea.git
cd revolu-idea
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# set keys (examples)
export TAVILY_API_KEY="your_key"
export EXA_API_KEY="your_key"

python main.py "What are the latest developments in quantum computing?"

Outputs

Reports are written under output/reports/ (gitignored). You can also find checkpoints and graph artifacts in output/.
