Quick start

Clone, install, set keys, run:

git clone https://github.com/jayhemnani9910/revolu-idea.git
cd revolu-idea
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

export TAVILY_API_KEY="your_key"
export EXA_API_KEY="your_key"

python main.py "What are the latest developments in quantum computing?"

No-key smoke run: LLM_PROVIDER=mock SEARCH_PROVIDER=mock

What it does

  • Plans a research strategy for your question.
  • Searches and gathers sources across multiple providers.
  • Extracts key claims and synthesizes findings.
  • Runs verification/audit loops to reduce hallucinations.
  • Writes Markdown reports under output/reports/.

Workflow (high level)

  → planner
  → web search (Tavily + Exa)
  → analysis + knowledge building
  → report generation (Markdown)
  → audit/verification loop
  → final report
